# 贡献指南

感谢您对农村互助养老积分超市管理系统的关注！

## 如何贡献

### 报告问题

1. 检查问题是否已被报告
2. 提供详细的问题描述
3. 提供复现步骤
4. 提供环境信息（操作系统、Node版本等）

### 提交代码

1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/amazing-feature`)
3. 提交更改 (`git commit -m 'Add amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建 Pull Request

### 代码规范

- 使用 ESLint 进行代码检查
- 遵循现有的代码风格
- 添加必要的注释
- 编写单元测试

### 提交信息规范

```
类型: 简短描述

详细说明（可选）

Fixes #123
```

类型包括：
- feat: 新功能
- fix: 修复
- docs: 文档
- style: 格式
- refactor: 重构
- test: 测试
- chore: 构建/工具

## 开发环境

```bash
# 后端
cd backend
npm install
npm run dev

# 前端
cd frontend
npm install
npm run dev
```

如有问题，欢迎提交 Issue 或 Pull Request。
